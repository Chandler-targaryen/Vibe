ğŸ§© 7. How Frontend and Backend Connect
âœ” Frontend never calls backend directly

Instead, Nginx handles:

Frontend â†’ Nginx â†’ Backend

For example:

Frontend sends:

POST http://localhost:3000/auth/login


Nginx rewrites internally to:

http://backend:8000/auth/login


So the frontend never needs the backend IP.

This makes deployment easier and removes CORS issues.
